Input System

The input system must maintains the following criteria:

  • Do not enter a protein sequence in fasta format
  • Uniprot Accession Number is optional
  • The sequence should contain at least one Lysine (K) residue
  • Only the 20 standard amino acid symbols and and one dummy amino acid sample 'X'are supported
  • The minimum length of protein sequence must be 21 residues

Output System

The prediction output consists of four items:

  • The protein fragment
  • Lysine position
  • Prediction decision

Browser Tool

The website has been tested with browsers of Firefox 43.0.4 or later.
